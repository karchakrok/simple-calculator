# simple-calculator
C'est un petit projet web rapide pour tester Flask. L'application permet de se connecter et d'accéder à une calculatrice basique codée en JavaScript.

Pour le lancer, rien de compliqué, il suffit d'avoir Python puis :

1.  **Installez Flask** (si ce n'est pas déjà fait) :
    ```bash
    pip install flask
    ```

2.  **Lancez l'application** :
    ```bash
    python views.py
    ```

3.  Ouvrez votre navigateur sur `http://localhost:5000`.

Les identifiants sont hardcodés et admin:admin marche.
